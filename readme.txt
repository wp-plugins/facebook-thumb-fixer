=== Facebook Thumb Fixer ===
Contributors: mikeyott
Tags: facebook, thumb, fixer, default, thumbnail, thumbnails, thumbs, og:image, open, graph
Requires at least: 2.9.2
Tested up to: 3.5
Stable tag: trunk

Fixes the problem of the missing (or wrong) thumbnail when a post is shared on Facebook.

== Description ==

This plugin is for those who have the problem where sharing a post on Facebook shows the wrong (or no) thumbnail image.

== Installation ==

Install, activate, done.

== Uninstall ==

Deactivate the plugin, delete if desired.
